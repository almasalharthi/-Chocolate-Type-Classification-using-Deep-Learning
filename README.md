# 🍫 Chocolate Type Classification using Deep Learning

> Can Artificial Intelligence recognize chocolate simply by looking at it?

## 📖 Project Overview

This project demonstrates how Deep Learning can classify different chocolate types from images using Computer Vision techniques.

The model analyzes visual patterns and predicts the chocolate category with high accuracy.

This project was developed as part of the AI and Robotics training program.

---

## 🎯 Objectives

- Build an image classification model.
- Classify different chocolate types.
- Apply Deep Learning techniques.
- Evaluate prediction performance.
